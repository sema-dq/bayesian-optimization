# Bayessche Optimierung — Interaktive Demo

Interaktives Tool zur Visualisierung der Bayesschen Optimierung.

**Kurs:** Optimierung Verfahren  
**Autor:** Simon Gwangwaa

## Features

- Schritt-für-Schritt GP-Visualisierung mit Confidence-Band
- Einstellbare Kernel, Acquisition Functions und Parameter
- Vergleich: BO vs Random Search vs Grid Search
- Hyperparameter-Tuning eines Neural Networks
- Theorie-Seite mit kompakter Erklärung

## Live Demo

```
pip install -r requirements.txt
streamlit run app.py
```
